Security Practices

  • All data encrypted in transit via HTTPS/TLS
  • Data encrypted at rest (Supabase AES-256)
  • 4-digit PIN login with brute-force lockout (no password reuse risk)
  • Session tokens expire automatically after 30 days
  • Row-Level Security (RLS) — drivers can only see their own data
  • Stripe PCI DSS Level 1 compliance for payment data
  • No third-party advertising SDKs

🗑️ Request Data Deletion

If you want your personal data deleted from 3 Lakes Logistics systems, follow these steps:

Option 1 — Email Request

  1. Send an email to dispatch@threelakeslogistics.com
  2. Subject line: Data Deletion Request
  3. Include your name and phone number on file
  4. We will confirm deletion within 30 days

Option 2 — In-App Request

  1. Open the 3 Lakes Driver app
  2. Go to Profile tab
  3. Scroll to bottom → tap "Delete My Account"
  4. Confirm deletion — account removed immediately
