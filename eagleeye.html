Dispatcher Checklist

☀️ Morning (8–10AM)
Review all active loads — check statuses
Check-call all drivers currently on load
Search DAT + Truckstop for available freight
Book loads for all available carriers
Send rate confirmations for new loads
Review expiring COIs / insurance alerts
🌤️ Midday (11AM–2PM)
Second round check-calls (all active loads)
Follow up on any delayed or exception loads
Collect PODs from completed loads
Submit invoices for delivered loads
Book tomorrow's loads (plan 1 day ahead)
Coordinate fuel advances if needed
🌙 Close (3–6PM)
Final check-calls for all active loads
Log all load updates in board
Follow up on outstanding invoices
Review DAT for next-day opportunities
Update dispatch board for tomorrow
Record daily revenue & load count
📅 Weekly Tasks
Review RPM averages per carrier
Follow up on all open AR invoices
Post in Facebook owner-operator groups
Cold outreach: 5 new carriers (DAT database)
Review upcoming insurance / permit expirations
📆 Monthly Tasks
Send RPM report to all carriers
Generate IFTA mileage summary (Full-Service)
Review P&L in QuickBooks
Upsell check-in: offer Full-Service upgrade
Lane analysis: identify highest RPM corridors
🏳️ Quarterly / Annual
File IFTA quarterly return (Apr/Jul/Oct/Jan)
Renew UCR registration (annual)
File 2290 HVUT (due August 31)
Annual DOT inspection for Truck #1
Review insurance policies for renewal
